Documentation:

'airpassengers' is a shiny web application that is taking data from 'AirPassengers' dataset in the datasets package.
It can show the monthly airline passenger numbers from year 1949 to 1960.

'airpassengers' come with a few tab panels and a few basic widgets;

  • Plot
    • Selectbox - 'Year', to select desired year.
    • Single check box - 'Show All Year', is an option to show all year.
    • Radiobutton - 'Plot type', is an option to display the data either using a bar or line plot.
  • Table
    • Table of MonthlyAirline Passengers data from 1949 to 1960
  • Documentation

By default, the application will display data based on preselected value, which is Year: 1949 and Plot type: bar.
Once the user choose to change the input values from the available widgets, the display chart will changed accordingly.
Be reminded to untick 'Show all year' if need the chart to display as single year data.
